Features

Platform > DevOps

Getting your code from dev to production has never been smoother. git push origin big-onion from a zsh prompt is all you need.

We'll check your tests, and if everything looks good, update your running instances with your new code. Easy peasy.

Convention > Frustration

Enjoy developing applications again, using a fully integrated (and highly opinionated) platform based on the Chicago Boss framework.

We know there are other ways to do it, but we'd rather not spend the time worrying about them.

Fault Tolerance > Scalability

Scalability is easy, but fault tolerance is what you really want. Our homogenous app nodes marry each app instance with a Riak node.

We'll make sure that your app continues to serve traffic and your data is safe.
